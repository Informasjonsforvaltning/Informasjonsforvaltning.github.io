Required software

In your linux machine you will need to install some software

Docker

Instructions here

Docker compose

Instructions here

Maven

% sudo apt-get install maven

OpenJDK

% sudo apt-get install default-jdk

This is as of now openjdk-11. If you need java 8, do

% sudo apt-get install openjdk-8-jdk

Kotlin

% snap install kotlin

Git

% sudo apt-get install git

Python

Python (both 2.7 and 3.) should be preinstalled.

% python --version
Python 2.7.16+
% python3 --version
Python 3.7.4

NodeJS

It is highly recommended to use the node version manager (nvm). To install nvm, follow instructions.

Then, to download, compile, and install the latest release of node, do this:

% nvm install node # "node" is an alias for the latest version
